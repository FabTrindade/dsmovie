# DS Movie. 



This is a project of a web site for movie reviews.

*Backend in Java with Spring Tool Suite 4 (STS).
*Frontend with React.
*Cloud hosting using Netify and Heroku (with PostgreSQL database)

Link: https://fabsdsmovie.netlify.app/
